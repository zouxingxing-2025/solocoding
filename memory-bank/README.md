# 记忆库 (Memory Bank)

本目录存放项目的关键决策记录、架构演化历史和技术选型理由。

## 目录结构

- `决策记录.md` - 重要技术选型、架构变更的决策背景和理由
- `演化历史.md` - 项目各版本的主要变更摘要
- `待解决问题.md` - 已知但暂未修复的问题或未来优化方向

## 使用说明

- 在每次重大决策后（如技术栈选型、模块拆分、插件选择），AI 应更新 `决策记录.md`。
- 在每个阶段完成后，AI 可更新 `演化历史.md`。
- 用户可手动添加 `待解决问题.md` 中的条目，AI 会定期检查。

## 注意

- 本目录随项目代码一起提交，不自动清理。
- 决策记录应包含：日期、决策内容、理由、备选方案、最终选择。
