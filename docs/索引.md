# 根索引 - AI 编程组件包

## 强制规则（AI 必须遵守）
1. **在加载任何其他文档之前，必须先读取 `.ai/进度状态.json` 和 `AGENTS.md` 的核心章节（第0-1章）**。
2. 所有操作必须严格遵循 `AGENTS.md` 中的约束，不得偏离。
3. 以下阶段必须按顺序执行。完成一个阶段后，更新 `.ai/进度状态.json` 并加载下一阶段索引。

## 加载策略
- **初始状态**：`.ai/进度状态.json` 中 `current_phase = "0"`, `current_step = "P1"`（如果文件不存在，则创建）。
- **步骤循环**：
  1. 根据 `current_step` 加载对应的技能文件（`.trae/skills/` 下的 `.md` 文件）。
  2. 执行技能中的指令。
  3. 步骤完成后，更新 `.ai/进度状态.json` 中的 `current_step` 为下一步。
  4. 若步骤需要人工干预（如确认数据模型），暂停并等待用户输入，恢复后继续。
- **阶段切换**：当一个阶段的所有步骤完成后，更新 `current_phase`，进入下一阶段。

## 人工干预点（共5处）

## 技能文件说明
- 所有技能文件位于 `.trae/skills/` 目录，按照阶段和步骤编号命名。
- 每个技能文件包含前置条件、目标、Action、Assert、Artefact、Analysis、Adjust、Advance、干预等部分。
- AI 执行时需严格遵循技能文件的指令，不得偏离。
