# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `P1`（或项目未初始化）。
- 如果进度状态文件不存在，则视为初始状态。

## 目标
解压用户提供的组件包到项目根目录，确保目录结构完整。

## Action
- 检查当前目录是否为空（允许存在 `.trae` 和 `.ai` 目录）。
- 如果为空，提示用户上传组件包压缩文件（.zip 或 .tar.gz），或提供下载链接。
- 使用系统命令解压（`unzip` 或 `tar -xzf`）到当前目录。
- 解压后，验证关键文件存在：`AGENTS.md.template`、`.trae/skills/` 目录。

## Assert
- 目录为空或用户确认覆盖。
- 解压命令执行成功。
- 关键文件存在。

## Artefact
- 项目根目录下出现组件包的所有文件。

## Analysis
- 如果目录非空但用户确认覆盖，则备份原有文件到 `.ai/backup/`。
- 如果解压失败，分析错误原因（权限、磁盘空间、压缩包损坏）。

## Adjust
- 若解压失败，提示用户手动解压并提供指导。
- 若缺少关键文件，输出错误报告并终止步骤。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "P2",
    "steps_completed": ["P1"]
  }
  ```
- 提示用户：下一步应执行 P2。

## 干预
需要用户提供压缩包路径或确认覆盖非空目录。


## When to Use This Skill
Use this skill when P1_解压组件包 is required, such as during phase P tasks.

## Not For
- This skill does not handle other phases or unrelated tasks.
