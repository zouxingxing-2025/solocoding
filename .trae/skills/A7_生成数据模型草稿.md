# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `A7`。
- `A6` 已完成。

## 目标
基于提取的实体、字段和关系，生成初始的 `data-model.json` 草稿文件。

## Action
- 读取 `.ai/进度状态.json` 中的 `entities`、`fields`、`relations`。
- 组装成符合 `shared/schemas/data-model.schema.json` 的 JSON 对象。
- 为每个实体添加 `$schema` 引用和元数据（如 `title`、`description`）。
- 写入临时文件 `.ai/tmp/data-model-draft.json`。

## Assert
- 生成的 JSON 格式有效，且通过 schema 验证。
- 所有实体至少有一个主键字段（标记 `"primary": true`）。

## Artefact
- `.ai/tmp/data-model-draft.json`

## Analysis
- 检查是否有实体缺少主键，自动添加 `id` 字段（string, required, primary）。
- 检查关系引用是否正确（外键字段的类型必须与目标实体的主键类型匹配）。

## Adjust
- 如果验证失败，修正字段类型或关系定义后重新生成。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "A8",
    "steps_completed": ["...", "A7"]
  }
  ```
- 提示用户：下一步应执行 `A8`。

## 干预
- 无。


## When to Use This Skill
Use this skill when A7_生成数据模型草稿 is required, such as during phase A tasks.

## Not For
- This skill does not handle other phases or unrelated tasks.
