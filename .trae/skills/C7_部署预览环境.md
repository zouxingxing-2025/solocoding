# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `C7`。
- `C6` 已完成。

## 目标
为每个候选demo部署临时预览环境（本地端口或临时URL），供用户试用。

## Action
- 读取 `.ai/demos/demo_list.json` 获取所有候选demo。
- 对于每个demo：
  - 进入demo目录，启动开发服务器（如 `npm run dev` 或 `python app.py`）。
  - 分配可用的本地端口（或使用TRAE的临时端口转发功能）。
  - 记录访问URL（如 `http://localhost:3001`）。
- 生成预览环境列表，存入 `.ai/demos/preview_urls.json`。
- 如果部署失败，重试最多2次。

## Assert
- 每个demo至少有一个可访问的URL。
- 服务在30秒内启动成功。

## Artefact
- `.ai/demos/preview_urls.json`（包含demo名称和URL）。

## Analysis
- 检查端口冲突，自动切换。
- 若部署超时，终止该demo的部署并记录错误。

## Adjust
- 如果某个demo无法部署，跳过该demo并在列表中标记"部署失败"。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "C7.5",
    "steps_completed": ["...", "C7"]
  }
  ```
- 提示用户：下一步应执行 C7.5（集成测试）。

## 干预
无（自动部署）。


## When to Use This Skill
Use this skill when C7_部署预览环境 is required, such as during phase C tasks.

## Not For
- This skill does not handle other phases or unrelated tasks.
