# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `P4`。
- `P3` 已完成（TRAE 已启动并加载根索引）。

## 目标
读取全局经验库（`~/.ai_experience/`）的索引文件，但不加载具体经验内容。将索引信息（技术栈、领域、标签等）缓存到本地，供后续步骤按需查询。

## Action
- 检查全局经验库路径（默认 `~/.ai_experience/`，若环境变量 `AI_EXPERIENCE_PATH` 存在则使用该路径）。
- 如果路径不存在，创建目录并初始化空索引 `index.json`。
- 读取 `index.json`，获取经验条目摘要（ID、类型、标签、技术栈、领域、验证次数）。
- 将索引内容缓存到 `.ai/经验库缓存.json`（仅索引，不包含具体内容）。

## Assert
- 全局经验库目录存在或成功创建。
- `index.json` 文件存在且格式正确（若为空，则创建默认结构）。

## Artefact
- `.ai/经验库缓存.json`（包含索引摘要）。

## Analysis
- 如果索引损坏，尝试从备份恢复（`~/.ai_experience/.backup/`）。
- 如果全局经验库版本过旧，提示用户升级。

## Adjust
- 若无法读取，暂时跳过经验库功能，并记录警告。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "A1",
    "steps_completed": ["P1", "P2", "P2.1", "P2.2", "P2.3", "P3", "P4"]
  }
  ```
- 提示用户：准备阶段完成，下一步应执行 A1（引导用户描述工作流程）。

## 干预
无（全自动）。
