# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `P2.3`。
- `P2.2` 已完成。

## 目标
验证沙箱内 SSH 免密克隆。

## Action
- 启动容器，执行 `ssh -T git@github.com`。
- 或克隆一个测试仓库（例如 `git clone git@github.com:example/private-repo.git /tmp/test-clone`）。

## Assert
- 连接成功，无需密码。

## Artefact
- 验证日志（`.ai/logs/ssh_verify.log`）。

## Analysis
- 若失败，检查 socket 路径和私钥。

## Adjust
- 回退到 P2.1 或 P2.2。

## Advance
- 清理临时文件，加载 P3。

## 干预
- 无。
