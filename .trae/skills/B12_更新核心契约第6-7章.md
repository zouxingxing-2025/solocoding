# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `B12`。
- `B11` 已完成。

## 目标
将功能模块清单和插件配置写入核心契约 `AGENTS.md` 的第6章和第7章。

## Action
- **首先**：调用 `创建备份分支` 技能（除非用户明确要求 `--no-backup`）。
- 等待备份完成后，再执行以下操作：
- 读取 `AGENTS.md` 当前内容。
- 读取 `.ai/config/modules.yaml` 和 `.ai/config/plugins.yaml`。
- 更新第6章"功能模块清单"：根据 `modules.yaml` 生成表格（模块ID、名称、需求文档路径、状态）。
- 更新第7章"插件配置"：根据 `plugins.yaml` 生成 API 插件和 UI 插件的配置摘要（如使用哪些插件、关键参数）。
- 保存更新后的 `AGENTS.md`。

## Assert
- 第6章和第7章的内容已填充，无占位符残留。
- 表格格式正确，插件配置清晰。

## Artefact
- 更新后的 `AGENTS.md`。

## Analysis
- 检查模块清单是否与之前锁定的章节（第2-5章）引用一致。
- 检查插件配置是否与生成的代码匹配（如插件名称、版本）。

## Adjust
- 如果发现不一致（例如某个模块在 `modules.yaml` 中但代码未生成），标记警告并提示用户。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "B13",
    "steps_completed": ["...", "B12"]
  }
  ```
- 提示用户：下一步应执行 B13。

无。
