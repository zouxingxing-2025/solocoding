# 前置条件
- `.ai/进度状态.json` 中的 `current_step` 必须为 `P2.1`。
- `P2` 已完成。

## 目标
检测主机 SSH Agent 是否运行，并确认已添加至少一个私钥。

## Action
- 执行命令检测 SSH Agent：
  - macOS/Linux: `ssh-add -l`
  - Windows: `Get-Service ssh-agent`（检查状态）
- 如果命令失败（退出码非0），判断是 Agent 未运行还是无私钥。
- 根据结果向用户输出指导信息：
  - 若 Agent 未运行：提供启动命令（macOS/Linux: `eval "$(ssh-agent -s)"`；Windows: `Set-Service -StartupType Automatic; Start-Service ssh-agent`）
  - 若 Agent 运行但无私钥：提示用户运行 `ssh-add ~/.ssh/id_rsa`（或指定私钥路径）

## Assert
- SSH Agent 进程存在。
- 至少有一个私钥已添加。

## Artefact
- 检测结果记录到 `.ai/进度状态.json`，新增字段 `ssh_agent_ready: true/false`。

## Analysis
- 如果用户多次失败，分析可能原因：未生成 SSH 密钥、权限问题、环境变量未设置。

## Adjust
- 若用户无法立即配置，可跳过此步骤（标记 `ssh_agent_ready: false`），后续步骤需用户手动输入密码。

## Advance
- 更新 `.ai/进度状态.json`：
  ```json
  {
    "current_step": "P2.2",
    "steps_completed": ["P1", "P2", "P2.1"]
  }
  ```
- 提示用户：下一步应执行 P2.2。

## 干预
用户需确认已执行启动命令或添加私钥。


## When to Use This Skill
Use this skill when P2.1_检测SSH_Agent is required, such as during phase P tasks.

## Not For
- This skill does not handle other phases or unrelated tasks.
